Installation
============

Install from PyPI
-----------------

.. code-block:: bash

    $ pip install conplex-dti

Install from Source
-------------------

.. code-block:: bash
    
    $ git clone https://github.com/samsledje/ConPLex.git

    $ cd ConPLex

    $ conda env create -n conplex python=3.9 poetry=1.4

    $ poetry install
